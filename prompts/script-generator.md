# Full UGC Ad Script Generator Prompt

## Purpose
Generate complete, ready-to-shoot UGC ad scripts with timing, visual cues, and dialogue.

## Prompt Template

```
You are a senior UGC content strategist at a performance marketing agency.

CONTEXT:
- Brand: [BRAND NAME]
- Product: [PRODUCT NAME]
- Price: [PRICE]
- Target Audience: [AUDIENCE DESCRIPTION]
- Main Pain Point: [PAIN POINT]
- Key USP: [UNIQUE SELLING PROPOSITION]
- Platform: [PLATFORM - Instagram Reels / YouTube Shorts / Paid Ads]
- Script Format: [FORMAT - Honest Review / Before-After / Myth Buster / GRWM / Story Arc / Quick Tips]
- Tone: [TONE - Friendly / Excited / Calm / Bold / Storytelling]

TASK:
Generate a complete UGC video ad script following the PASC framework:
P - Problem (agitate the pain)
A - Agitation (make it worse)
S - Solution (introduce product naturally)
C - Call to Action (single clear action)

STRUCTURE (include for each section):
1. HOOK (0-3 seconds) — The scroll-stopper
2. PROBLEM (3-8 seconds) — Describe the pain in viewer's own words
3. SOLUTION (8-20 seconds) — Introduce product like telling a friend
4. PROOF (20-25 seconds) — Show results, social proof
5. CTA (25-30 seconds) — Single, clear action with urgency

FOR EACH SECTION INCLUDE:
- Exact dialogue (spoken text)
- Visual direction (what to show on screen)
- Timing (seconds)
- Tone notes

RULES:
- Total length: 25-40 seconds
- Write like a real person sharing with friends
- NO marketing speak. NO "revolutionary". NO "game-changer"
- Include sensory details (how it feels, smells, looks)
- One clear CTA only — don't overwhelm
- Include 1-2 text overlays per script
```

## Example Output: "The Honest Review" for GlowNaturals

### Script: The Honest Review
- **Duration:** 30 seconds
- **Platform:** Instagram Reels
- **Tone:** Friendly & Casual

---

**HOOK (0-3s)**
> "Okay so I've been using this serum for 2 weeks and I need to talk about it…"

📹 *Close-up selfie, natural lighting, speaking directly to camera*

---

**PROBLEM (3-8s)**
> "So my skin has been SO dull lately. Like, I tried everything — expensive brands, DIY masks, even drinking more water. Nothing was working."

📹 *Casual setting, bedroom or bathroom mirror*

---

**SOLUTION (8-18s)**
> "Then my friend told me about GlowNaturals Vitamin C Serum. It's all natural — no parabens, no chemicals — and it's only ₹599. I was skeptical but I tried it anyway."

📹 *Show the product, applying it, close-up of the bottle label*

---

**PROOF (18-25s)**
> "Look at my skin right now. No filter, no editing. This is literally 14 days of using it every night. My dark spots are fading and my skin just… glows."

📹 *Close-up of glowing skin, turn head to show different angles, no filter*

---

**CTA (25-30s)**
> "Link in bio — they have 20% off right now. Honestly at ₹599 it's already a steal. Try it. Your skin will thank you 💛"

📹 *Point down to the bio, smile, tap the link gesture*

# Brand Tone Adapter Prompt

## Purpose
Adapt any existing UGC script to match a different brand's voice, audience, and product while keeping the UGC authenticity.

## Prompt Template

```
You are a brand voice consultant specializing in UGC content.

ORIGINAL SCRIPT:
[PASTE YOUR EXISTING UGC SCRIPT HERE]

NEW BRAND DETAILS:
- Brand Name: [NEW BRAND]
- Industry: [INDUSTRY]
- Target Audience: [AUDIENCE]
- Brand Voice: [VOICE DESCRIPTION - e.g., "luxury and sophisticated" or "fun and youthful"]
- Product: [PRODUCT]
- Price Point: [PRICE]
- Key Differentiator: [WHAT MAKES IT UNIQUE]

TASK:
Adapt the script to match the new brand while keeping the UGC authenticity.

ADAPT THESE ELEMENTS:
1. Vocabulary (match the audience's language)
2. Energy level (calm for luxury, energetic for Gen-Z)
3. Pain points (specific to this product/audience)
4. Social proof references (relevant to this industry)
5. CTA style (match the platform and brand)

PRESERVE:
- First-person perspective
- Natural, unscripted feel
- Problem-solution-CTA structure
- Real human emotion
- Specific details (not vague claims)

OUTPUT:
- Adapted full script with timing
- 3 alternative hooks
- 2 CTA options
- Visual direction notes
```

## Example: Adapting from Skincare to Fitness

### Original (GlowNaturals - Skincare):
> "Okay so I've been using this serum for 2 weeks and I need to talk about it… My skin was SO dull. Then I found GlowNaturals Vitamin C Serum — all natural, ₹599 — and my skin literally glows now."

### Adapted (FitFuel - Protein Powder):
> "Okay so I switched protein powders 3 weeks ago and I have to say something… I was SO bloated with my old protein. Then my gym buddy told me about FitFuel — clean ingredients, no artificial sweeteners, ₹899 — and my workouts have never been better."

### Key Adaptations Made:
| Element | Skincare | Fitness |
|---------|----------|---------|
| Pain Point | Dull skin | Bloating from protein |
| Social Proof | "My friend told me" | "My gym buddy" |
| USP | All natural | Clean ingredients |
| Result | Skin glows | Better workouts |
| Vocabulary | Serum, glow, skin | Protein, gains, workouts |
| Tone | Casual female | Energetic, active |

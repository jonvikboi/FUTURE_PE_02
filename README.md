## 🎨 Ad Framework Used

Every script follows the **PASC Framework**:

```
P — Problem    → Agitate the pain point
A — Agitation  → Make it worse, make viewer feel seen
S — Solution   → Introduce product naturally
C — CTA        → Single clear action with urgency
```

### Script Structure:
| Section | Timing | Purpose |
|---------|--------|---------|
| **Hook** | 0-3s | Stop the scroll |
| **Problem** | 3-8s | Describe pain in viewer's words |
| **Solution** | 8-20s | Introduce product like telling a friend |
| **Proof** | 20-25s | Show results, social proof |
| **CTA** | 25-30s | One clear action |

## 🚀 How to Use

### For Your Own Brand:
1. Open the [Prompt Templates](#prompts) section
2. Copy the prompt template you need
3. Replace the `[PLACEHOLDER]` values with your brand details
4. Paste into ChatGPT, Claude, or Gemini
5. Use the generated content for your ads

### With the Live Generator:
1. Scroll to the [Generator](#generator) section
2. Fill in your brand details
3. Select platform, tone, and format
4. Click "Generate" to create a custom script
5. Copy the output and use it directly
